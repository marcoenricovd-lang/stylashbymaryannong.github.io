# Stylash by Mary Ann Ong

A premium beauty booking system website built with Next.js, featuring:

- **Landing Page** - Beautiful showcase of services and portfolio
- **Online Booking** - Full booking flow with calendar and payment
- **Admin Dashboard** - Manage bookings, calendar, settings, and overbooking

## Features

- Lash Extensions & Brow Services
- Nail Artistry
- Hair Coloring & Treatments
- Semi-Permanent Makeup (SPMU)
- Training & Certification

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- bcryptjs for authentication
- date-fns for date handling
- lucide-react for icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Admin Access

Navigate to `/admin/login` to access the admin dashboard.

## Deployment

This site is configured for GitHub Pages deployment.
